# Nytrsa Creator HQ 🎮

Personal stream dashboard for Nytrsa — built for Thu/Fri/Sat streams at 7PM EST.

## Features
- Live clock & date
- Weather for Lehigh Acres FL + Atlanta GA
- Quick links to all socials
- Daily content plan checklist
- Stream time cues with live active highlighter
- Last stream analytics + AI Stream Coach
- Twitter/X news feed

## Deploy
This site is deployed via **GitHub Pages**.

To update: edit `index.html` and push to the `main` branch. Changes go live automatically.
